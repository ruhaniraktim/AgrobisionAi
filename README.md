# 🌾 AgroVision AI

AgroVision AI is an AI-based crop disease detection system designed to help farmers identify plant diseases using image analysis.

## 💡 Features
- Upload leaf image
- Detect crop disease
- Provide Bangla suggestions
- Simple farmer-friendly UI
- Offline demo support

## 🎯 Purpose
To reduce crop loss in Bangladesh by providing fast and easy disease detection support to farmers.

## 🚀 Future Improvements
- Real AI model integration
- Voice assistant
- Weather-based alerts
- Mobile app version
